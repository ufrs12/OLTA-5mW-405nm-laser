### Стоковое использование
На пружинку подаем GND, на корпус +5В. Настраиваем фокус широкой отверткой. Шлиц под отвертку, как будто не настолько глубокий, чтобы отвертка могла поцарамать линзу. Если не хочется рисковать, то можно использовать какой-нибудь "рогатый" инструмент.
